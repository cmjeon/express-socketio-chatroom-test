## 설치

```bash
$ mkdir express-socketio-chatroom-test

$ cd express-socketio-chatroom-test

# npm 초기화
$ npm init

# 기본 modules 설치
$ npm install express socket.io moment nodemon
```

## 실행하기

### 기본적인 node.js 실행 

```bash
# 기본 실행
$ node app.js

# nodemon 설치 및 실행 : 변경 시 서버 재시작
$ npm install -g nodemon
$ nodemon app.js
```
